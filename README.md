# homebrew-gvm

Homebrew tap for [GVM (Go Version Manager)](https://github.com/ibreez3/gvm).

## Installation

```bash
brew tap ibreez3/gvm
brew install gvm
```

## Usage

After installation, initialize gvm:

```bash
gvm init
source ~/.zshrc  # or source ~/.bashrc
```

## Upgrade

```bash
brew upgrade gvm
```

## Uninstall

```bash
brew uninstall gvm
brew untap ibreez3/gvm
```

## Documentation

For more information about gvm, visit [https://github.com/ibreez3/gvm](https://github.com/ibreez3/gvm).
